Создай структуру проекта для веб-приложения "ВИ МПИИ" — веб-интерфейс для создания музыкальных плейлистов с применением технологии искусственного интеллекта. Проект называется Playlist_AI.

Стек: Python, FastAPI, Uvicorn, Jinja2 (серверный рендеринг), SQLAlchemy (async), PostgreSQL, Alembic (миграции).

Создай следующую структуру папок и файлов в текущей директории проекта:

app/
├── __init__.py
├── main.py                  # точка входа FastAPI, подключение роутеров
├── config.py                # загрузка переменных из .env (pydantic-settings)
├── database.py              # подключение к PostgreSQL, async engine, session
├── models/
│   ├── __init__.py
│   ├── user.py              # модель пользователя
│   ├── playlist.py          # модель плейлиста
│   └── track.py             # модель трека
├── routers/
│   ├── __init__.py
│   ├── auth.py              # маршруты регистрации и авторизации
│   ├── playlist.py          # маршруты генерации и управления плейлистами
│   └── pages.py             # маршруты для отдачи HTML-страниц
├── services/
│   ├── __init__.py
│   ├── ai_service.py        # взаимодействие с API нейросети
│   └── music_service.py     # взаимодействие с API музыкального сервиса
├── schemas/
│   ├── __init__.py
│   ├── user.py              # Pydantic-схемы для пользователя
│   └── playlist.py          # Pydantic-схемы для плейлиста
├── templates/
│   ├── base.html            # базовый шаблон с header/footer
│   ├── index.html           # главная страница
│   ├── login.html           # страница входа
│   ├── register.html        # страница регистрации
│   ├── generate.html        # страница генерации плейлиста
│   └── profile.html         # профиль с историей плейлистов
└── static/
    ├── css/
    │   └── style.css
    └── js/
        └── main.js

Также в корне проекта создай:
├── alembic.ini
├── .env.example
├── .gitignore
├── requirements.txt
└── README.md

В requirements.txt добавь:
fastapi
uvicorn[standard]
jinja2
sqlalchemy[asyncio]
asyncpg
alembic
pydantic-settings
python-dotenv
passlib[bcrypt]
python-jose[cryptography]
httpx

В .env.example добавь:
DATABASE_URL=postgresql+asyncpg://user:password@localhost:5432/playlist_ai
SECRET_KEY=your-secret-key
AI_API_KEY=your-ai-api-key
AI_API_URL=https://api.example.com
MUSIC_API_KEY=your-music-api-key
MUSIC_API_URL=https://api.example.com

В config.py реализуй класс Settings через pydantic-settings, загружающий все переменные из .env.

В database.py создай async engine и async sessionmaker для SQLAlchemy.

В main.py создай экземпляр FastAPI, подключи роутеры, настрой Jinja2Templates и StaticFiles.

Все остальные файлы (модели, роутеры, сервисы, схемы) создай с заглушками и комментариями о том, что будет реализовано позже.

Инициализируй git-репозиторий. Виртуальное окружение .venv уже создано — не трогай его.
